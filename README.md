### Bewegen clone site
- 프로젝트 소요기간: 10일
- HTML, CSS, JavaScript를 활용하여 Bewegen 클론 사이트를 만들어 보았습니다.
- 메인 페이지와 nav 메뉴 까지 오리지널 웹사이트의 동적인 효과를 최대한 똑같이 구현하기 위해 노력했습니다.


- 클론 사이트 주소: https://ene8255.github.io/bewegen-clone/
- original 사이트 주소: https://bewegen.com/en
